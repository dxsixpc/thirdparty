## API

```typescript
// 加密
cryptoBase64.encrypt(value: string);
// 解密
cryptoBase64.decrypt(value: string);
```

### 参数

| 参数  | 说明               | 类型   |
| ----- | ------------------ | ------ |
| value | 需要加密解密的数据 | string |

### 返回

(string):返回加密或解密过的数据。
